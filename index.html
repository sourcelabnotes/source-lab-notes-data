Source Ledger Public Data

Download the scorecards and audit files behind Source Ledger, a public experiment in testing whether crypto clues predicted later movement.

These files are research evidence, not investment advice. We publish normalized observations and scores, not raw third-party dashboards or datasets.

Boundary

This repository is intended to make Source Ledger claims auditable. It should not contain raw screenshots, raw HTML, private dashboard exports, credentials, local databases, or account information.

Source Ledger publishes clues, scorecards, and caveats. Crypto is the first lab; the broader project is scoring prediction evidence.
